# AI-docs

> 🤖 AI学习文档与深度思考的知识库

## 📖 仓库简介

这个仓库是我在AI领域学习和研究过程中的文档记录与深度思考的集合。旨在通过系统性的学习、实践和总结，构建一个全面的AI知识体系。

### 🎯 主要目的

- **知识沉淀**：记录AI学习过程中的重要概念、理论和技术细节
- **深度思考**：对AI技术发展脉络进行分析和思辨
- **实践总结**：整理实际项目中的经验和教训
- **技术追踪**：跟进AI领域的最新发展和突破
- **知识分享**：为AI学习者提供参考和启发

## 📚 内容结构

### 当前研究主题

#### 🔄 [RNN → Transformer → Mamba 序列建模演进](./rnn_transformer_mamba/)
- **RNN基础理论**：循环神经网络的数学原理与梯度问题
- **Transformer革命**：自注意力机制与并行化突破
- **Linear Attention**：效率优化的关键技术
- **Mamba创新**：状态空间模型的新突破
- **对比分析**：四种架构的深度比较与适用场景

#### 🌍 [世界模型 (World Models) 深度学习](./world_models/)
- **基础理论**：概率图模型、状态空间模型、深度生成模型
- **经典架构**：World Models、PlaNet、Dreamer系列演进
- **强化学习融合**：Model-Based RL、Dyna架构、探索策略
- **视频生成**：视频预测、Sora、Genie等前沿模型
- **前沿应用**：自动驾驶、机器人学习、具身智能
